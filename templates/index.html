{% trans %}About{% endtrans %}

{% trans %}SymPy is a Python library for symbolic mathematics. It aims to become a full-featured computer algebra system (CAS) while keeping the code as simple as possible in order to be comprehensible and easily extensible. SymPy is written entirely in Python.{% endtrans %}

{% trans %}Why SymPy{% endtrans %}

{% trans %}SymPy is…{% endtrans %}

  • {% trans %}Free:{% endtrans %} {% trans %} Licensed under BSD, SymPy is free both as in speech and as in beer. {% endtrans %}
  • {% trans %}Python-based:{% endtrans %} {% trans %} SymPy is written entirely in Python and uses Python for its language. {% endtrans %}
  • {% trans %}Lightweight:{% endtrans %} {% trans %} SymPy only depends on mpmath, a pure Python library for arbitrary floating point arithmetic, making it easy to use. {% endtrans %}
  • {% trans %}A library:{% endtrans %} {% trans %} Beyond use as an interactive tool, SymPy can be embedded in other applications and extended with custom functions. {% endtrans %}

{% trans %}Projects using SymPy{% endtrans %}

{% trans %}This is an (incomplete) list of projects that use SymPy. If you use SymPy in your project, please let us know on our mailinglist, so that we can add your project here as well.{% endtrans %}

  • {% trans %}Cadabra{% endtrans %}: {% trans %} A system for Tensor algebra and Quantum Field Theory that uses SymPy for scalar algebra. {% endtrans %}
  • {% trans %}ChemPy{% endtrans %}: {% trans %} A package useful for chemistry written in Python. {% endtrans%}
  • {% trans %}devito{% endtrans %}: {% trans %} A symbolic DSL and just-in-time compiler for high performance stencil computation. {% endtrans%}
  • {% trans %}EinsteinPy{% endtrans %}: {% trans %} A Python package for symbolic and numerical General Relativity. {% endtrans%}
  • {% trans %}galgebra{% endtrans %}: {% trans %} Symbolic Geometric Algebra/Calculus package for SymPy.. {% endtrans%}
  • {% trans %}LaTeX Expression project{% endtrans %}: {% trans %} Easy LaTeX typesetting of algebraic expressions in symbolic form with automatic substitution and result computation). {% endtrans%}
  • {% trans %}Lcapy{% endtrans %}: {% trans %} Experimental Python package for teaching linear circuit analysis. {% endtrans%}
  • {% trans %}OctSymPy{% endtrans %}: {% trans %} A Symbolic Package for Octave using SymPy. {% endtrans %}
  • {% trans %}Optlang{% endtrans %}: {% trans %} A Python package for solving mathematical optimization problems. {% endtrans %}
  • {% trans %}PyDy{% endtrans %}: {% trans %} Multibody dynamics in Python. {% endtrans%}
  • {% trans %}pyneqsys{% endtrans %}: {% trans %} Solve symbolically defined systems of non-linear equations numerically. {% endtrans%}
  • {% trans %}pyodesys{% endtrans %}: {% trans %} Straightforward numerical integration of ODE systems from Python. {% endtrans%}
  • {% trans %}PyTorch TorchInductor{% endtrans %}: {% trans %} TorchInductor uses SymPy to support dynamic shapes and strides.{% endtrans%}
  • {% trans %}QMCPACK{% endtrans %}: {% trans %} Quantum Monte Carlo in C++. Sympy is used to generate reference values for unit tests and some code generation.{% endtrans%}
  • {% trans %}Quantum Programming in Python{% endtrans %}: {% trans %} Quantum 1D Simple Harmonic Oscillator and Quantum Mapping Gate. {% endtrans%}
  • {% trans %}SageMath{% endtrans %}: {% trans %} Open source mathematics system that includes SymPy. {% endtrans %}
  • {% trans %}Scikit-fdiff{% endtrans %}: {% trans %} Finite differences discretization. {% endtrans %}
  • {% trans %}SfePy{% endtrans %}: {% trans %} Simple finite elements in Python. {% endtrans%}
  • {% trans %}Spyder{% endtrans %}: {% trans %} The Scientific Python Development Environment, a Python equivalent to Rstudio or MATLAB; full SymPy support can be enabled in Spyder's IPython Consoles.{% endtrans%}
  • {% trans %}Symbolic statistical modeling{% endtrans %}: {% trans %} Adding statistical operations to complex physical models. {% endtrans%}
  • {% trans %}yt{% endtrans%}: {% trans %} Python package for analyzing and visualizing volumetric data (unyt, the yt unit system, uses SymPy). {% endtrans%}
