{% trans %}About{% endtrans %}

{% trans %}SymPy is a Python library for symbolic mathematics. It aims to become a full-featured computer algebra system (CAS) while keeping the code as simple as possible in order to be comprehensible and easily extensible. SymPy is written entirely in Python and does not require any external libraries.{% endtrans %}

{% trans %}Features{% endtrans %}

{% trans %}SymPy core capabilities include:{% endtrans %}

  • {% trans %}basic arithmetics{% endtrans %} *,/,+,-,**
  • {% trans %}basic simplification{% endtrans %} ({% trans %}like{% endtrans %} a*b*b + 2*b*a*b -> 3*a*b**2)
  • {% trans %}expansion{% endtrans %} ({% trans %}like{% endtrans %} (a+b)**2 -> a**2 + 2*a*b + b**2)
  • {% trans %}functions{% endtrans %} (exp, ln, ...)
  • {% trans %}complex numbers{% endtrans %} ({% trans %}like{% endtrans %} exp(I*x).expand(complex=True) -> cos(x)+I*sin(x))
  • {% trans %}differentiation{% endtrans %}
  • {% trans %}taylor (laurent) series{% endtrans %}
  • {% trans %}substitution{% endtrans %} ({% trans %}like{% endtrans %} x -> ln(x), {% trans %}or{% endtrans %} sin -> cos)
  • {% trans %}arbitrary precision integers, rationals and floats{% endtrans %}
  • {% trans %}noncommutative symbols{% endtrans %}
  • {% trans %}pattern matching{% endtrans %}

{% trans %}Then there are SymPy modules for these tasks:{% endtrans %}

  • {% trans %}more functions (sin, cos, tan, atan, asin, acos, factorial, zeta, legendre){% endtrans %}
  • {% trans %}limits{% endtrans %} ({% trans %}like{% endtrans %} limit(x*log(x), x, 0) -> 0)
  • {% trans %}integration using extended Risch-Norman heuristic{% endtrans %}
  • {% trans %}polynomials (division, gcd, square free decomposition, groebner bases, factorization){% endtrans %}
  • {% trans %}solvers (algebraic, difference and differential equations, and systems of equations){% endtrans %}
  • {% trans %}symbolic matrices (determinants, LU decomposition...){% endtrans %}
  • {% trans %}mpmath (multiprecision floating-point arithmetic){% endtrans %}
  • {% trans %}geometric algebra (GA){% endtrans %}
  • {% trans %}Pauli and Dirac algebra{% endtrans %}
  • {% trans %}quantum physics{% endtrans %}
  • {% trans %}geometry module{% endtrans %}
  • {% trans %}plotting (2D and 3D){% endtrans %}
  • {% trans %}code generation (C, Fortran, LaTeX){% endtrans %}
