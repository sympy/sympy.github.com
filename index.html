Beschreibung

SymPy ist eine Python-Library für symbolische Mathematik. Ziel des Projekts ist es, ein komplettes komplettes Computeralgebra-System (CAS), mit möglichst einfach gehaltenem Code, zu entwickeln, sodass es später leicht verständlich und gut erweiterbar ist. SymPy ist vollständig in Python geschrieben und benötigt keine externen Libraries.

Features

Die SymPy-Basis enthält folgende Funktionen:

  • Einfache Arithmetik *,/,+,-,**
  • Grundlegende Vereinfachungen (wie a*b*b + 2*b*a*b -> 3*a*b**2)
  • Ausklammern (wie (a+b)**2 -> a**2 + 2*a*b + b**2)
  • Funktionen (exp, ln, ...)
  • Komplexe Zahlen (wie exp(I*x).expand(complex=True) -> cos(x)+I*sin(x))
  • Differenzieren
  • Taylorreihen (Laurent-Reihen)
  • Substitution (wie x -> ln(x), oder sin -> cos)
  • Ganze, rationale und Gleitkommazahlen beliebiger Genauigkeit
  • Nichtkommutative Symbole
  • Pattern Matching

Außerdem gibt es SymPy-Module für folgende Aufgaben:

  • Weitere Funktionen (sin, cos, tan, atan, asin, acos, factorial (Fakultät), zeta, legendre)
  • Grenzwerte (wie limit(x*log(x), x, 0) -> 0)
  • Integration unter Verwendung erweiterter Risch-Norman-Heuristik
  • Polynome (Division, größter gemeinsamer Teiler, wurzelfreie Zerlegung, Gröbnerbasis, Faktorisierung)
  • Gleichungsauflöser (algebraisch, Differenzen- und Differentialgleichungen, Gleichungssysteme)
  • Symbolische Matrizen (Determinante, LU-Zerlegung ...)
  • mpmath (multiprecision floating-point arithmetic)
  • Geometrische Algebra (GA)
  • Pauli- und Dirac-Algebra
  • Quantenphysik
  • Geometrie-Modul
  • Plotten (2D und 3D)
  • Codeerzeugung (C, Fortran, LaTeX)
