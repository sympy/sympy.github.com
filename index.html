O SymPy

SymPy je knihovna jazyka Python pro symbolickou matematiku. Má za cíl stát se kompletním počítačovým algebraickým systémem (CAS, Computer algebra system) a zároveň být co nejjednodušší, aby byla pochopitelná a jednoduše rozšiřitelná. SymPy je napsaná kompletně v jazyce Python a nepotřebuje žádné externí knihovny.

Vybavení

Základní funkce SymPy zahrnují:

  • základní aritmetiku *,/,+,-,**
  • základní zjednodušování (jako a*b*b + 2*b*a*b -> 3*a*b**2)
  • roznásobování (jako (a+b)**2 -> a**2 + 2*a*b + b**2)
  • funkce (exp, ln, ...)
  • complexní čísla (jako exp(I*x).expand(complex=True) -> cos(x)+I*sin(x))
  • derivování
  • taylorova (laurentova) řada
  • substituce (jako x -> ln(x), nebo sin -> cos)
  • libovolná přesnost celých čísel, zlomků a čísel s plovoucí desetinnou čárkou
  • nekomutativní symboly
  • porovnávání vzorků

Pak má SymPy moduly pro tyto úlohy:

  • více funkcí (sin, cos, tan, atan, asin, acos, faktoriál, zeta, legendre)
  • limity (jako limit(x*log(x), x, 0) -> 0)
  • integrování s využitím rozšířeného Rich-Normanova algoritmu
  • polynomy (dělení, nsd, bezčtvercová dekompozice, gröbnerova báze, faktorizace)
  • řešiče (algebraické řešiče, derivace, diferenciální rovnice a soustavy rovnic)
  • symbolické matice (determinanty, LU dekompozice...)
  • mpmath (multiprecizní aritmetika s plovoucí desetinnou čárkou)
  • geometrická algebra (GA)
  • Pauliho and Diracova algebra
  • kvantová fyzika
  • geometrické moduly
  • grafy (2D a 3D)
  • generování kódu (C, Fortran, LaTeX)
