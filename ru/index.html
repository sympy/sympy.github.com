О SymPy

SymPy - библиотека Python для символических вычислений. Она претендует на звание полнофункциональной системы компьютерных расчётов (CAS), но при этом сохраняя простоту кода и расширяемость. Она полностью написана на Python и не требует других внешних библиотек.

Функции

Базовые возможности SymPy включают:

  • базовую арифметику *,/,+,-,**
  • базовые упрощения (такие, как a*b*b + 2*b*a*b -> 3*a*b**2)
  • раскрытие выражений (такие, как (a+b)**2 -> a**2 + 2*a*b + b**2)
  • функции (exp, ln, ...)
  • операции с комплексными числами (такие, как exp(I*x).expand(complex=True) -> cos(x)+I*sin(x))
  • дифференциацию
  • ряды Тейлора (Лорана)
  • подстановки (такие, как x -> ln(x), или sin -> cos)
  • операции с целыми и действительными числами произвольной точности
  • некоммутативные выражения
  • проверку соответствия шаблону

Модули SymPy включают поддержку следующего:

  • больше функций (sin, cos, tan, atan, asin, acos, factorial, zeta, legendre)
  • лимиты (такие, как limit(x*log(x), x, 0) -> 0)
  • интегрирование с использованием алгоритмов Рича-Нормана
  • полиномы (деление, НОД, свободное квадратное разложение, базис Грёбнера, факторизация)
  • решение уравнений (алгебраические, дифференциальные и их системы)
  • символические матрицы (определители, LU-разложения...)
  • вычисления большой точности для чисел с плавающей точкой
  • геометрическая алгебра
  • алгебра Поля и Дирака
  • квантовая физика
  • модуль геометрии
  • двух- и трёхмерное постороение графиков
  • генерация кода (C, Fortran, LaTeX)
