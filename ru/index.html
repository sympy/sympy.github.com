О SymPy

SymPy - библиотека Python для символьных вычислений. Она призвана стать полнофункциональной системой компьютерных расчётов (CAS), но при этом сохраняет простоту кода и расширяемость. Она полностью написана на Python и не требует других внешних библиотек.

Функции

Базовые возможности SymPy включают:

  • базовую арифметику *,/,+,-,**
  • базовые упрощения (такие, как a*b*b + 2*b*a*b -> 3*a*b**2)
  • раскрытие выражений (такие, как (a+b)**2 -> a**2 + 2*a*b + b**2)
  • функции (exp, ln, ...)
  • операции с комплексными числами (такие, как exp(I*x).expand(complex=True) -> cos(x)+I*sin(x))
  • дифференцирование
  • ряды Тейлора (Лорана)
  • подстановки (такие, как x -> ln(x), или sin -> cos)
  • операции с целыми и действительными числами произвольной точности
  • некоммутативные выражения
  • проверку соответствия выражения шаблону

Модули SymPy включают поддержку следующего:

  • дополнительные функции (sin, cos, tan, atan, asin, acos, factorial, zeta, legendre)
  • пределы (такие, как limit(x*log(x), x, 0) -> 0)
  • интегрирование с использованием алгоритмов Рича-Нормана
  • полиномы (деление, НОД, разложение, свободное от квадратов; базис Грёбнера, факторизация)
  • решение уравнений (алгебраические, разностные, дифференциальные и их системы)
  • символьные матрицы (определители, LU-разложения...)
  • вычисления с произвольной точностью для чисел с плавающей точкой
  • геометрическая алгебра
  • матрицы Паули и Дирака
  • квантовая физика
  • модуль геометрии
  • двух- и трёхмерное постороение графиков
  • генерация кода (C, Fortran, LaTeX)
