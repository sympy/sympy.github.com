关于

SymPy是一个符号数学Python库。它的目标是成为一个全功能的计算机代数系统，同时保持代码的精简而易于理解和课扩展。SymPy完全由Python写成，不需要任何外部库。

Google Summer of Code (GSoC) 2013

SymPy是GSoC 2013的一个指导组织。 点 这里 查看被接受的计划的链接与信息。

特性

核心能力

  • 基本算术： 支持如+, -, *, /, ** (幂)等运算符
  • 化简 三角函数, 多项式
  • 展开： 多项式
  • 函数： 三角函数, 双曲函数, 指数, 根式, 对数, 绝对值, 球谐函数, 阶乘与伽马函数, zeta函数, 多项式, 特殊函数……
  • 代换： 例子
  • 数： 任意精度整数/a>, 有理数 和浮点数
  • 非交换符号
  • 模式匹配

多项式

  • 基本算术： 除法，最大公因式……
  • 因式分解
  • 无平方分解
  • Gröbner基
  • 部分分式分解
  • 结式

微积分

  • 极限： limit(x*log(x), x, 0) -> 0
  • 微分
  • 积分： 它使用了扩展Risch-Norman启发算法
  • 泰勒(洛朗)级数

解方程

  • 多项式方程
  • 代数方程
  • 微分方程
  • 差分方程
  • 方程组

组合数学

  • 排列
  • 组合
  • 分拆
  • 子集
  • 置换群: Polyhedral, Rubik, Symmetric, ...
  • Prufer与格雷码

离散数学

  • 二项式系数

  • 积
  • 数论： 产生素数，素性测试，整数分解，……
  • 逻辑表达式

矩阵

  • 行列式
  • 转置
  • 求解
