À propos

SymPy est une bilibothèque Python pour les mathématiques symboliques. Elle vise à devenir un système de calcul formel complet tout en gardant le code aussi simple que possible afin d'être compréhensible et facilement extensible. SymPy est écrite entièrement en Python et ne requiert aucune bibliothèque externe.

Fonctionnalités

Les fonctionnalités de base de SymPy incluent:

  • l'arithmétique de base *,/,+,-,**
  • les bases de la simplification (comme a*b*b + 2*b*a*b -> 3*a*b**2)
  • le développement (comme (a+b)**2 -> a**2 + 2*a*b + b**2)
  • les fonctions (exp, ln, ...)
  • les nombres complexes (comme exp(I*x).expand(complex=True) -> cos(x)+I*sin(x))
  • la dérivation
  • les séries de taylor (laurent)
  • la substitution (comme x -> ln(x), ou sin -> cos)
  • les entiers, rationnels et flottants à précision arbitraire
  • les symboles non commutatifs
  • la correspondance de motifs

Les modules SymPy suivants sont disponibles pour ces tâches:

  • plus de fonctions (sin, cos, tan, atan, asin, acos, factorielle, zeta, legendre)
  • limites (comme limit(x*log(x), x, 0) -> 0)
  • intégration utilisant l'algorithme étendu de Risch-Norman
  • polynômes (division, pgcd, décomposition sans facteur carré, bases de Gröbner, factorisation)
  • résolutions (algébriques, équations différentielles, et systèmes d’équations)
  • matrices symboliques (déterminants, décomposition LU...)
  • mpmath (arithmétique flottante multiprécision)
  • algèbre géométrique
  • Algèbre de Pauli et Dirac
  • physique quantique
  • module de géométrie
  • traçage (2D et 3D)
  • génération de code (C, Fortran, LaTeX)
