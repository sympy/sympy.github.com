Sobre

SymPy é uma livraria Python para matemática simbólica. Almejamos nos tornar um sistema de algebra computacional (CAS) completo mantendo o código o mais simples possível para ser compreensível e facilmente extendido. SymPy é escrito inteiramente em Python e não requer nenhuma outra livraria.

Recursos

Capacidades do Core

  • Aritimética básica Suporte de operadores como +, -, *, /, ** (potência)
  • Simplificação
  • Expansão
  • Funções: trigonométrica, hiperbólica, exponencial, raízes, logarítmos, módulo harmônicas, fatoriais e funções gamma, zeta, polinômios, funções especiais, ...
  • Substituição
  • Números inteiro de precisão arbitrária, racionais e decimais
  • Símbolos não comutativos
  • Detecção de padrão

Polinomiais

  • Aritimética básica divisão, mdc
  • Fatoração
  • Decomposição de raíz
  • Bases de Gröbner
  • Decomposição de frações parciais
  • Resultantes

Cálculo

  • Limites: limit(x*log(x), x, 0) -> 0
  • Diferenciação
  • Integração: Usa heurística de Risch-Norman extendida
  • Séries de Taylor (Laurent)

Resolução de equações

  • Equações polinomiais

  • Equações diferenciais
  • Relação de recorrência
  • Sistemas de equações
