Acerca de

SymPy es una biblioteca de Python para matemática simbólica. Apunta a convertirse en un sistema de algebra total (CAS) mientras mantiene el código tan simple como es posible para que sea comprensible y fácilmente extensible. SymPy está escrito totalmente en Python y no requiere bibliotecas adicionales.

Google Summer of Code (GSoC) 2013

SymPy es una organización aceptada en GSoC 2013. Aplica ahora, el último dia es el 2 de Mayo, 2013. Mira el anuncio en G+ Para links e información de Cómo aplicar.

Características

Capacidades principales

  • Aritmética básica Soporte para operadores tales como +, -, *, /, ** (potencia)
  • Simplificación
  • Expansión
  • Funciones: trigonometría, hipérbolas, exponencial, raíces, logaritmos, valor absoluto, armónicos esféricos, factoriales y funciones gamma, funciones zeta, polinomios, funciones especiales, ...
  • Sustitución
  • Números: integers de precisión arbitraria, racionales, y floats
  • Símbolos no-conmutativos
  • Comparación de patrones

Polinomios

  • Aritmética básica división, mcd
  • Factorización
  • Descomposición libre de cuadrados
  • bases Gröbner
  • Descomposición de fracciones parciales
  • Resultantes

Cálculo

  • Límites: limit(x*log(x), x, 0) -> 0
  • Derivadas
  • Integrales: Usa la heurística extendida de Risch-Norman
  • Series de Taylor(Laurent)

Resolución de ecuaciones

  • Ecuaciones polinomiales
  • Ecuaciones algebraicas
  • Ecuaciones Diferenciales
  • Relaciones de recurrencia
  • Sistemas de ecuaciones

Combinatorias

  • Permutaciones
  • Combinaciones
  • Particiones
  • Subconjuntos
  • Grupos de Permutación: Polyhedral, Rubik, Symmetric, ...
  • Códigos Prufer y Gray

Matemática discreta

  • Coeficientes Binomiales
  • Sumatorias
  • Productorias
  • Teoría de números: generación de números primos, pruebas de primalidad, factorización de integers, ...
  • Expresiones lógicas

Matrices

  • Aritmética básica
  • Eigenvalores/eigenvectores
  • Determinantes
  • Inversión
  • Solución
  • Expresiones abstractas

Álgebra Geométrica

Geometría

  • puntos, lineas, rayos, segmentos, elipses, círculos, polígonos, ...
  • Intersección
  • Tangentes
  • Similaridad

Gráficas

  • Modos de coordenadas
  • Gráficas de entidades geométricas
  • 2D y 3D
  • Interfaz interactiva
  • Colores

Física

  • Unidades
  • Mecánica
  • Cuántico
  • Óptica de Gauss
  • Álgebra de Pauli

Estadística

  • Distribución Normal
  • Distribución uniforme
  • Probabilidades

Escritura

  • Escritura mejorada: Escritura mejorada en ASCII/Unicode, LaTeX
  • Generación de Código: C, Fortran, Python
