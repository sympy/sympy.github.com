Acerca de

SymPy es una biblioteca de Python para matemática simbólica. Apunta a convertirse en un sistema de algebra total (CAS) mientras mantiene el código tan simple como es posible para que sea comprensible y fácilmente extensible. SymPy está escrito totalmente en Python y no requiere bibliotecas adicionales.

Características

Capacidades principales

  • Aritmética básica Soporte para operadores tales como +, -, *, /, ** (potencia)
  • Simplificación Trigonometry, Polynomials
  • Expansion: of a polynomial
  • Funciones: trigonometric, hyperbolic, exponential, roots, logarithms, absolute value, spherical harmonics, factorials and gamma functions, zeta functions, polynomials, special functions, ...
  • Substitution: example
  • Números: arbitrary precision integers, rationals, and floats
  • Símbolos no-conmutativos
  • Comparación de patrones

Polinomios

  • Aritmética básica división, mcd
  • Factorización
  • Descomposición libre de cuadrados
  • bases Gröbner
  • Descomposición de fracciones parciales
  • Resultantes

Cálculo

  • Límites: limit(x*log(x), x, 0) -> 0
  • Derivadas
  • Integrales: Usa la heurística extendida de Risch-Norman
  • Series de Taylor(Laurent)

Resolución de ecuaciones

  • Ecuaciones polinomiales
  • Ecuaciones algebraicas
  • Ecuaciones Diferenciales
  • Relaciones de recurrencia
  • Sistemas de ecuaciones

Combinatorias

  • Permutaciones
  • Combinaciones
  • Particiones
  • Subconjuntos
  • Grupos de Permutación: Polyhedral, Rubik, Symmetric, ...
  • Códigos Prufer y Gray

Matemática discreta

  • Coeficientes Binomiales
  • Sumatorias
  • Productorias
  • Teoría de números: generación de números primos, pruebas de primalidad, factorización de integers, ...
  • Expresiones lógicas
