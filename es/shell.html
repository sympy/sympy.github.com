SymPy Live

Instructions

Press ENTER to run the code or use the Run button in the toolbar.

Note it can take up to 30 seconds before the shell finishes loading and is ready to run commands.

 

About this page

SymPy Live shell allows you to use SymPy in the browser without the need to install anything on your computer.

The following commands are executed by default: 

[1]: from sympy import *
     x, y, z, t = symbols('x y z t')
     k, m, n = symbols('k m n', integer=True)
     f, g, h = symbols('f g h', cls=Function)

SymPy Live shell is powered by JupyterLite. It can take up to 30 seconds before the shell code and libraries load completely and become available for using interactively.

Example session

[2]: expr = (x + y)**3

[3]: expand(expr)
[3]: \(x^{3} + 3 x^{2} y + 3 x y^{2} + y^{3}\)

[4]: sin(x).series(x, 0, 7)
[4]: \(x - \frac{x^{3}}{6} + \frac{x^{5}}{120} + O\left(x^{7}\right)\)

Read the tutorial in the SymPy docs to learn more about SymPy: SymPy Tutorial.
